Module 02 Assignment

Complete the following problems:

  1. Construct the truth table for (A and B) or (A and C)
  2. Construct the truth table for (not A) or B
  3. You go to the restaurant and the server asks you if you want the soup or the salad with your entree. Is "or" in this situation the same as logical or? Why or why not?

For the following use trinket blocks:

  1. Create a program that asks the user for their name, then prints out their name the same number of times as letters in the name.

    For example, if the user entered bob then your program should print 
    What is your name? bob
bob
bob
bob
    Problem 4 Template
  2. Create a program that asks the user for a letter continually and prints it out unless they enter a vowel. If the user enters a vowel, print "you entered a vowel" and end the program.

    Here is example output: 
    What is your letter? p
p
What is your letter? r
r
What is your letter? q
q
What is your letter? a
"You entered a vowel!"
    Problem 5 Template
  3. Create a program that asks the user for a number, the prints out whether that number is even 
    What is your number? 27
This is an odd number.
    Problem 6 Template
  4. Create a program that calculates the mean from a list of numbers (you must use the loop over the list block). The list is set up for you in the template.
    Problem 7 Template (Important)
  5. Create a program that asks for a score between 0 and 100, then prints out the letter grade (A, B, C, D, F) 
    What is your score? 87
You got a B!
    Problem 8 Template

BONUS (not required):

  1. Create a program which prints out the prime numbers under 100 (must calculate whether prime as the program runs, not compare to a list of known prime numbers)
  2. Print the sum of the prime numbers under 1000

Due Sunday, April 2nd at 11:59 pm
