From the variable menu, drag the set variable block to your canvas.
Drag the type of value you want to set to the canvas and snap it to the set variable block
Fill in the value for in the value block by typing in the small box.
To use the variable, drag the variable block with the connector on the left to your canvas.
Snap the variable block into the block you want it to interact with.
